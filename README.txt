===============> Sistema de Cadastro de Produtos <===============

Projeto desenvolvido em TypeScript com o objetivo de criar um sistema simples de cadastro e visualização de produtos utilizando armazenamento local no navegador.

< Sobre o Projeto >

O sistema permite que o usuário realize login, cadastre produtos e visualize os produtos cadastrados. Os dados são armazenados utilizando o `localStorage`, permitindo que permaneçam salvos mesmo após fechar o navegador.

O projeto foi desenvolvido para praticar conceitos básicos de TypeScript, manipulação do DOM e armazenamento de informações no navegador.

===============> Funcionalidades <===============

- Login de usuário
- Cadastro de produtos
- Armazenamento dos produtos no navegador
- Visualização dos produtos cadastrados
- Navegação entre páginas do sistema

===============> Tecnologias Utilizadas <===============

- HTML5
- CSS3
- TypeScript
- JavaScript
- LocalStorage

===============> Como Executar o Projeto <===============

1. Clone o repositório:

```bash
git clone https://github.com/heliosenai-dev/Cadastro-de-Produtos.git
```

2. Instale o TypeScript caso não tenha:

```bash
npm install -g typescript
```

3. Compile os arquivos TypeScript:

```bash
tsc
```

4. Abra o arquivo `index.html` no navegador.

===============> Funcionamento <===============

O usuário realiza o login pelo sistema. Após entrar, pode acessar a página de cadastro e adicionar novos produtos.

Os produtos cadastrados são convertidos para JSON e armazenados no `localStorage`. Depois, a página de produtos recupera essas informações e exibe os dados na tela.

## Objetivo

Este projeto tem como objetivo aplicar conhecimentos iniciais de TypeScript, como:

- Tipagem de dados
- Criação de funções
- Manipulação de elementos HTML
- Organização de arquivos
- Conversão de TypeScript para JavaScript
